# Tax calculator module
    There are 2 parts to this project:
    UI : ReactJS
    Server : ExpressJS
    to run this project locally

    ### UI : 
        ```
        cd ui
        yarn install
        yarn start
        ```
    ### server : 
        ```
        cd server
        yarn install
        node index.js
        ```
    

    The UI will run on port 3000 
    The server will run on port 8000
